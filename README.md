# Ninja Bomber

[Live Site](https://www.jjyang.me/ninja_bomber)

Ninja Bomber is a 2-player game inspired by NES Bomberman and Naruto. This project is built on HTML5, HTML5 Canvas, CSS3, and vanilla JavaScript.
